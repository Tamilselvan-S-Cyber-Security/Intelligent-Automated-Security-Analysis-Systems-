# Intelligent Automated Security Analysis System

**Project Title:** Intelligent Automated Security Analysis System for Applications, Websites, and Files

**Product/Tool Name:** CyberWolfScanner

**Domain:** Cybersecurity / Web Application Security / Secure Software Engineering

**Developed by:** Security Team - SKP Engineering College, CSE 4th Year (Final Year Project)

**Technology Stack:**

- Python 3.x
- Flask (web server + REST endpoints)
- Streamlit (interactive dashboard UI)
- Requests + BeautifulSoup4 (web requests + HTML parsing)
- Plotly / Matplotlib / Seaborn (visual analytics)
- `dnspython`, `python-whois` (DNS + OSINT)
- `cryptography` (key management/encryption utilities)
- Optional: local scanning modules under `modules/`

**Repository (workspace):** `CyberWolfScanner`

**Main Entry Points:**

- Flask server: `main.py` -> `app.py`
- Streamlit UI: `wolf.py`
